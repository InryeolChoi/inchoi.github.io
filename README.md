# Inchoi Profile

React, TypeScript, Vite, Motion, and react-i18next based profile site for GitHub Pages.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Update content

Most editable content lives in `src/data/siteContent.ts`.

- `profile`: name, intro, email, location, profile image path
- `skillCategories`: grouped skills
- `timelineItems`: timeline entries
- `portfolioItems`: portfolio cards and GitHub links
- `projectItems`: main projects
- `experienceItems`: career section

Replace the placeholder email and experience entries with your real information.
